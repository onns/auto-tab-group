## 使用  
git 下载此项目 或者release中下载压缩包  
打开扩展页面的 开发人员模式  
选择加载解压缩的扩展  

## 网页使用
不会对已有的标签页进行整理  
请打开新标签页使用  
edge扩展安装:[标签页自动分组](https://microsoftedge.microsoft.com/addons/detail/%E6%A0%87%E7%AD%BE%E9%A1%B5%E8%87%AA%E5%8A%A8%E5%88%86%E7%BB%84/hifeijficcakiokpfabdflnjdbihofdd)
